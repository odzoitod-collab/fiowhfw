# Галерея этапов

Статический HTML-сайт с галереей изображений этапов.

## Структура проекта

```
├── index.html          # Главная страница с галереей
├── public/
│   └── stages/         # Папка с изображениями этапов
│       ├── stage1.jpg
│       ├── stage2.jpg
│       └── ...
├── vercel.json         # Конфигурация для Vercel
└── package.json        # Метаданные проекта
```

## Развертывание на Vercel

### Способ 1: Через Vercel CLI
1. Установите Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Войдите в аккаунт:
   ```bash
   vercel login
   ```

3. Разверните проект:
   ```bash
   vercel --prod
   ```

### Способ 2: Через GitHub
1. Создайте репозиторий на GitHub
2. Загрузите все файлы (включая изображения)
3. Подключите репозиторий к Vercel

### Устранение проблем с изображениями

Если изображения не отображаются:

1. **Проверьте, что все файлы загружены:**
   ```bash
   ls -la *.jpg
   ```

2. **Убедитесь, что изображения не слишком большие** (Vercel имеет лимит на размер файлов)

3. **Проверьте консоль браузера** на наличие ошибок 404

4. **Попробуйте принудительное развертывание:**
   ```bash
   vercel --prod --force
   ```

## Локальный запуск

```bash
npm install
npm start
```

Сайт будет доступен по адресу http://localhost:3000
